💍 Ali Weds Hira — Wedding Invitation Website

A cinematic and modern wedding invitation website featuring animated curtains, smooth transitions, and an interactive event reveal system. Designed to deliver a premium digital wedding experience.

⸻

✨ Features
	•	🎬 Curtain Opening Animation (Top + Left + Right)
	•	🔘 Frosted “Open” Button Interaction
	•	🎶 Background Music (Tap to Start)
	•	📅 Step-by-Step Event Reveal
	•	💎 Glassmorphism UI Design
	•	💡 Glow Effects & Smooth Animations
	•	📍 Google Maps Integration
	•	📱 Fully Responsive (Mobile Friendly)

🎵 Customization

Change Music

Replace the audio source in HTML:

<audio id="music" loop>
  <source src="your-music-link.mp3">
</audio>


⸻

Update Event Details

Edit event sections inside HTML:
	•	Mehndi
	•	Barat
	•	Walima

⸻

Update Locations

Replace Google Maps links in anchor tags:

<a href="YOUR_GOOGLE_MAP_LINK">Get Location</a>


⸻

Change Images

Replace curtain and credit images:

background:url('your-image-link')
